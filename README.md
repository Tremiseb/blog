# Projet PHP
Projet académique : Développement d'un blog en PHP
Dépôt Github utilisé : https://github.com/Tremiseb/blog

## Getting started
Lors de la création de compte le mdp est hasé donc nous vous invitons à créer votre propre compte admin
Veuillez créer un fichier config.ini similaire au fichier config.ini.example afin de rendre le site fonctionnel en local
Vous trouverez le script de création de la base de données ainsi que les inserts dans le fichier script_bd.txt
Bonne journée !
